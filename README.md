# STREETXWEAR

## Interacciones con el cliente

_Apuntando al directorio del cliente usando `cd client`._

### _**Iniciar cliente debug (Vitejs):**_

```
npm run dev
```

### _**Generar build:**_

```
npm run build
```

### _**Ejecución de la build local:**_

```
npm run preview
```

_Se requiere haber generado la build previamente._
